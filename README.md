# 📱 Mobile Sales Analytics Dashboard

A complete end-to-end Data Analytics project built using **Power BI** and **MySQL** to analyze mobile phone sales across multiple brands, cities, payment methods, and time periods.

## 📌 Project Overview

This project focuses on analyzing mobile phone sales transactions to derive meaningful business insights. The project includes:

- Interactive Power BI Dashboard
- SQL Analysis (Basic, Intermediate & Advanced Business Queries)
- Business Insights
- Power BI Report (.pbix)
- MySQL Analysis
- Dashboard Screenshots

---

## 📊 Dashboard Features
