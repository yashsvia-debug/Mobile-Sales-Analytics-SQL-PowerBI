📌 Project Overview



This project focuses on analyzing mobile phone sales transactions to derive meaningful business insights. The project includes:



\- Interactive Power BI Dashboard

\- SQL Analysis (25 Business Queries)

\- Business Insights

\- Power BI Report (.pbix)

\- Dashboard Screenshots



\---



🛠️ Tech Stack



\- Power BI, Power Query, DAX

\- MySQL

\- Microsoft Excel

\- Git \& GitHub



\---



📊 Dashboard Features



\- KPI Cards

\- Brand-wise Sales Analysis

\- City-wise Sales Analysis

\- Monthly Sales Trends

\- Payment Method Distribution

\- Customer Ratings Analysis

\- Interactive Slicers

\- Dynamic Filtering
